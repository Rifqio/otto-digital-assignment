
## Installation Guides Local
Install dependencies with npm

```bash
  npm install 
```

Run the dev server

```bash
  npm run dev
```

To run the unit test
```bash
  npm run test
```

Access the page on http://localhost:3000

## Installation Guides Docker

Run docker compose up on the terminal

```bash
  docker-compose up
```

Access the page on http://localhost:3000
